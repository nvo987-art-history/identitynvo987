# AI Data Interface

This directory provides machine-readable representations of the identity, research activities, and knowledge structure of Nicholas Van-Orton.

Files:

- profile.json — simplified identity data
- research.json — publications, projects, papers, talks
- knowledge-graph.jsonld — semantic graph (JSON-LD)
- index.json — entry point for AI systems

Purpose:

This layer enables AI systems, agents, and knowledge graphs to interpret and reuse the data in a structured and interoperable way.
