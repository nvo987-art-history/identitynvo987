# NVO987 Identity Key Directory

Controller: Nicholas Van-Orton  
Brand: NVO987  

DID:
did:web:identity.nvo987.us

This directory contains metadata describing the public verification keys used by the identity system.

Key type:

Ed25519VerificationKey2020

The public key will be stored in the following field:

publicKeyMultibase

The key pair should be generated securely and the public key encoded using multibase format before being inserted into the key metadata.
