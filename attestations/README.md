# DID Attestation Archive

This directory contains versioned cryptographic attestations of the DID document.

Each version represents a complete attestation package containing the DID document
and its associated OpenTimestamps proofs. These proofs demonstrate that the
document hash was submitted for timestamping and later anchored in the Bitcoin
blockchain.

The purpose of this archive is to maintain a transparent and verifiable history
of DID document versions together with their blockchain-based proof of existence.

Each version acts as an independent cryptographic record that can be verified
using the OpenTimestamps verification tools.
