# DID Attestation Version 1

This directory contains the first version of the DID document and its
associated OpenTimestamps proofs.

The included timestamp proofs demonstrate that the document hash was
submitted for timestamping and later anchored in the Bitcoin blockchain.

Files in this directory represent the original attestation package
for version 1 of the DID document.
